# 选题知识库 · 生活化版（含钩子设计）

## 一、选题钩子的底层逻辑

很多人以为爆款靠运气，其实不然。爆款内容是有规律可循的，掌握了公式，你也能批量生产爆款。**爆款公式：痛点选题+吸睛标题+黄金开头+干货内容+互动钩子**，这五个要素缺一不可。

### 1.1 什么是钩子

钩子就是在文章开头或标题中，用一些吸引人的语言或内容，激发用户的好奇心，让他们想要继续阅读下去。

### 1.2 钩子的8种类型

根据实战经验，钩子可以分为以下8类，每类都有对应的公式模板：

表格

| 钩子类型     | 核心逻辑                       | 公式模板                                       |
| ------------ | ------------------------------ | ---------------------------------------------- |
| **好奇类**   | 利用观众得不到、没体验过的事物 | “XX是一种什么体验？”“为什么其他人XX，却XX样？” |
| **借势类**   | 名人自带流量和热点             | “某明星都在用的XX”“某剧大火，我却关心里面的XX” |
| **痛点类**   | 痛点不解决就难受               | “为什么XX，却XX”“不知道XX？一定要看”           |
| **极限类**   | 将抽象内容具象化               | “这是全网最XX”“99%的人都不知道的XX”            |
| **恐吓类**   | 起到警示作用，引起危机感       | “XX揭秘，千万不要”“如果你再不XX，就会XX”       |
| **反差类**   | 利用前后对比，形成反差         | “你只告诉XX，却不知道”“我能XX，却不能”         |
| **利益类**   | 利益是每个人都最在乎的         | “没想到XX就可以”“分享一个月入XX的小项目”       |
| **同理心类** | 人之初，性本善                 | “我今天XX”“人生第一次XX，能支持下吗？”         |

## 二、生活化选题案例库（含钩子设计）

### 2.1 职场类选题

**选题1：面试谈薪水的技巧**

**钩子设计**：痛点类+好奇类

- 标题：“为什么你面试总是谈不到高薪水？”

- 开头钩子：“面试官问‘你期望薪资是多少’，你是不是直接报了个数字，然后就没了？其实，这个问题背后有3个陷阱，90%的人都踩过。”

- 内容方向：面试中谈薪水的具体话术、避坑指南

- 互动钩子结尾：“你面试时遇到过最无语的问题是什么？评论区告诉我”

**选题2：同事甩锅怎么办**

**钩子设计**：恐吓类+痛点类

- 标题：“同事又把锅甩给你？这3招让他哑口无言”

- 开头钩子：“你是不是也遇到过这种情况：明明是同事的错，最后背锅的却是你。别急，今天教你3句话，让甩锅的人当场闭嘴。”

- 内容方向：职场甩锅的应对话术、自我保护技巧

- 互动钩子结尾：“你还遇到过哪些职场‘坑’？评论区聊聊”

**选题3：35岁职场危机**

**钩子设计**：恐吓类+利益类

- 标题：“35岁被裁员后，我才明白这3个道理”
- 开头钩子：“去年这个时候，我被公司‘优化’了。35岁，房贷、车贷、孩子学费，一夜之间全压在身上。但正是这次失业，让我想明白了一些事……”
- 内容方向：职场危机的应对策略、副业思路、心态调整
- 互动钩子结尾：“你觉得35岁真的是职场分水岭吗？评论区说说你的看法”

## 三、选题钩子的黄金开头模板

根据实战经验，视频或文章的开头3秒至关重要。以下是8个火爆的开头文案模板：

表格

| 模板类型   | 公式                   | 案例                                           |
| ---------- | ---------------------- | ---------------------------------------------- |

## 四、选题的互动钩子设计

很多人忽略了互动钩子，但这是提升互动率的关键。在笔记末尾设置一个问题或者小任务，引导用户评论互动，能让互动率提升50%。

### 4.1 互动钩子模板

表格

| 类型       | 模板                               | 案例                                       |
| ---------- | ---------------------------------- | ------------------------------------------ |
| **提问类** | “你还遇到过哪些问题？评论区告诉我” | “你面试时遇到过最无语的问题是什么？”       |
| **求助类** | “你觉得XX应该怎么做？帮我出出主意” | “你觉得朋友借钱该不该借？帮我出出主意”     |
| **共鸣类** | “有没有人和我一样？”               | “有没有人和我一样，每次减肥都失败？”       |
| **投票类** | “你选A还是B？评论区告诉我”         | “你觉得该不该删掉前任的微信？评论区告诉我” |
| **挑战类** | “敢不敢在评论区立个flag？”         | “敢不敢在评论区立个flag：这个月瘦5斤？”    |

### 4.2 互动钩子的设置原则

1. **低门槛**：问题要简单，用户不需要思考太久
2. **有共鸣**：问题要能引发用户的共同经历或感受
3. **有话题性**：问题要有讨论空间，不是“是/否”就能回答的
4. **有奖励感**：让用户觉得“回答了就能得到什么”（如得到建议、被看见等）

## 五、选题的“抄”与“创”策略

新媒体运营是一门需要不断创新和学习的技能，而选题是新媒体运营的核心环节之一。一个好的选题，可以吸引更多的用户关注，提高阅读量和转化率，甚至成为爆款文章。

### 5.1 如何“抄”出爆款选题

不是让你抄袭，而是学习爆款选题的底层逻辑：

1. **拆解爆款结构**：找到爆款内容，分析它的选题角度、钩子设计、内容框架
2. **换场景应用**：把A领域的爆款选题，换到B领域（如把“职场甩锅”换成“家庭甩锅”）
3. **换人群应用**：把针对年轻人的选题，换成针对宝妈/学生/老年人的版本
4. **换平台应用**：把知乎的爆款问答，改成小红书/抖音的笔记形式

### 5.2 找选题的神器

找选题并不难，只要你掌握一些找选题的神器，就可以轻松获取热点和灵感：

- **搜索引擎**：百度、谷歌、必应等，利用自动补全功能获取选题方向
- **社交媒体**：微博热搜、抖音热榜、知乎热榜、小红书热门
- **数据工具**：5118、微指数、头条指数、Google Trends
- **问答平台**：知乎、百度知道、Quora，发现用户的真实疑问和痛点

## 六、选题自检清单

每确定一个选题，用以下清单自检：

```
□ 这个选题是否贴近普通人的生活？
□ 这个选题是否有明确的钩子（好奇/痛点/利益/恐惧）？
□ 这个选题是否有足够的目标受众？
□ 这个选题是否有独特的切入角度？
□ 这个选题是否具备传播性（用户愿意转发）？
□ 这个选题的互动钩子是否设计好了？
```

**判断标准**：

- 3个以下“是” → 选题需优化
- 3-4个“是” → 合格选题
- 5-6个“是” → 优质选题

## 七、总结：选题钩子的核心公式

**爆款选题 = 生活化场景 + 精准痛点 + 强力钩子 + 互动引导**

- **生活化场景**：让用户觉得“这说的就是我”
- **精准痛点**：击中用户最头疼的问题
- **强力钩子**：在开头3秒抓住用户注意力
- **互动引导**：让用户愿意评论、转发

掌握了这个公式，你也能批量生产爆款选题。
