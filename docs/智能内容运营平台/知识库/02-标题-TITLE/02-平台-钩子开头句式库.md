# 标题知识库 · 钩子句式库（标题 = 开头 = 前3秒）

## 一、标题在 ACS 里的作用

- **一键生成**：BODY 已合并标题，但标题库仍用于「标题重做」和钩子检索
- **核心原则**：标题不是文艺创作，是 **3 秒内让人点进来且不后悔** 的钩子
- **抖音**：标题与封面共同设定期望，须与前 3 秒口播一致
- **小红书**：标题兼 **搜索关键词**（前 18 字含 2 个核心词）

---

## 二、万能句式（复制改写）

```
[人群] + [痛点/场景] + [数字] + [结果/悬念]
```

### 痛点提问类

- 「你是不是也\_\_\_？」
- 「为什么你***，却总是***？」
- 「\_\_\_的人，一定懂这种崩溃」

### 结果前置类

- 「我用***，把***从 A 变成 B」
- 「**_天搞定_**，附步骤」
- 「坦白局：\_\_\_我只改了这一点」

### 反常识类

- 「别再***了，***才是正解」
- 「**_爆火，不是因为_**，而是因为\_\_\_」
- 「99% 的人不知道\_\_\_」

### 避坑恐吓类

- 「**_千万别先做_**」
- 「新手\_\_\_最容易踩的 3 个坑」
- 「不知道***，你会多花***小时」

### 数字清单类

- 「***个***技巧，第 2 个最狠」
- 「一张图看懂\_\_\_」
- 「**_步搞定_**，懒人必备」

## 四、标题 × 开头一致性

| 标题               | 开头第 1 句必须呼应 |
| ------------------ | ------------------- |
| 「8 分钟搞定周报」 | 立刻提 8 分钟或结果 |
| 「3 个避坑」       | 立刻提第 1 坑       |
| 「OpenClaw 实测」  | 画面/口播出现工具   |

标题党（标题承诺 > 前 3 秒交付）= 高点击、低完播、被减推。

---

## 五、A/B 标题测试

同一内容写 3 个标题：

- A 痛点提问
- B 结果前置
- C 反常识

小范围发布后，保留 **点击率 × 完读/完播** 综合最高的，不是只看点击。

---

## 六、避坑

- 「分享」「推荐」「干货」无具体利益 → 点击差
- 标题 30 字小作文 → 信息流展示不全
- 违禁：最、第一、100%、躺赚
- 标题与封面、开头各说各话 → 后悔阈值大
