# 封面知识库 · 小红书/抖音原生封面（拒绝商务风）

## 一、为什么你的封面「像 PPT / 像公司宣传」

AI 默认会画成：**蓝紫渐变、西装会议室、抽象科技线条、玻璃幕墙、SaaS 官网风**——这在信息流里像广告，用户本能划走。

**小红书/抖音爆款封面共性**（万赞笔记结构复盘）：

- 像 **真人博主做的笔记**，不像品牌投放
- **大字 + 少元素 + 强对比**，缩略图 0.5 秒能读懂
- **生活场景感**：书桌、咖啡、手机、便签、手写感
- 配色 **2～3 色**，奶油/莫兰迪/多巴胺，忌 corporate 蓝

IMAGE Agent 生图时必须走 **「笔记博主风」**，禁止商务风。

## 二、账号默认画风（AI 效率 / 干货向）

| 项     | 推荐                                   | 禁止                     |
| ------ | -------------------------------------- | ------------------------ |
| 主风格 | 大字报 / 备忘录 / 手绘信息图           | 商务简约、企业宣传       |
| 背景   | 奶油米白 #FFF8F0、浅杏、薄荷浅绿、浅粉 | 深蓝渐变、灰色 corporate |
| 强调色 | 番茄红、柠檬黄、珊瑚橙（1 个就够）     | 科技蓝 #2563EB 大面积    |
| 字体感 | 粗黑体 / 手写标题 / 便签马克笔感       | 细线商务黑、英文科技感   |
| 装饰   | emoji 贴纸、荧光笔划线、对话气泡       | 抽象几何、世界地图线框   |
| 人物   | 卡通打工人、背影、手部特写             | 西装微笑 stock 人像      |

## 三、四种高点击封面模板（优先用）

### 1. 纯字大字报（干货/避坑/清单）

- 3:4 竖版，背景 **纯色或轻微纸质纹理**
- 主标题占画面高度 **1/3 以上**，3～8 个字
- 1 个强调色块 + 1 个 emoji 点缀
- 例：米白底 + 黑色巨字「周报 8 分钟」+ 黄色高亮「AI」

### 2. 备忘录 / 便签风（教程/步骤）

- 仿 iPhone 备忘录、手账便签、格子纸
- 标题像 **刚打出来的两行字**，带下划线或荧光笔标记
- 适合 AI 工具、步骤类内容

### 3. 对比分屏（前后变化）

- 左「乱」右「顺」、左「40min」右「8min」
- 高饱和对比，但 **不要商务信息图图标**
- 用生活化场景：乱桌面 vs 清爽桌面

### 4. 真实场景 + 少字（生活感/口播向）

- 俯拍书桌、手机屏、咖啡杯、笔记本 **自然光**
- 图上 **≤7 字** 大字，其余靠标题
- 抖音封面：人脸表情或产品 **特写**，背景虚化

## 四、封面文案（叠在图上或留给后期）

```
封面字 = 钩子，不是摘要。主标题 ≤8 字。
```

| 类型   | 示例               |
| ------ | ------------------ |
| 数字   | 「3 招 8 分钟」    |
| 痛点   | 「写不出周报？」   |
| 反常识 | 「别教 AI 写周报」 |
| 身份   | 「打工人必看」     |

**字在图上的规则**：主标题用引号让生图模型渲染——「周报8分钟」；副标题更小，不超过一行。

## 五、平台规格

| 平台   | 比例             | 安全区              |
| ------ | ---------------- | ------------------- |
| 小红书 | 3:4（1728×2304） | 四角勿放关键信息    |
| 抖音   | 9:16 或 3:4 截帧 | 底部 25% 避开互动栏 |

封面利益点必须与 **前 3 秒口播/前 3 行正文** 一致。

## 六、Seedream 封面提示词模板（复制改写）

### 模板 A · 小红书大字报

```text
小红书爆款封面，竖版3比4，大字报风格，奶油米白色纸质背景，轻微纸张纹理。
画面中央超大粗黑手写感标题「周报8分钟」，其中「8分钟」用荧光黄高亮划线。
右下角一个小小卡通打工人贴纸和咖啡杯简笔画，emoji 风格，可爱不写实。
整体像普通博主自己做的笔记封面，生活化、接地气。
禁止：商务风、企业宣传、深蓝科技渐变、西装人物、会议室、stock photo、
玻璃幕墙、SaaS网页风、过多英文、复杂信息图表。
```

### 模板 B · 备忘录风

```text
小红书备忘录风格封面，竖版3比4，仿手机备忘录界面，浅黄便签纸背景。
顶部两行手写感标题「AI学会我操作了」「下周一键跑周报」，黑色字迹，一行荧光粉下划线。
周围少量手绘箭头和小星星贴纸，留白充足，清新可爱。
禁止：商务PPT、企业蓝、写实摄影棚、西装、科技感抽象线条。
```

### 模板 C · 抖音生活场景

```text
抖音封面竖版，真实感俯拍书桌场景，自然窗光，笔记本电脑和手机屏幕发光，
桌上有咖啡和便签纸，背景轻微虚化，生活Vlog质感。
画面上方叠加粗体白字黑描边「8分钟搞定周报」，字数少、对比强。
像博主自己拍的封面，不是广告海报。
禁止：商务宣传、企业展厅、过度HDR、假笑模特。
```

## 七、生图后自检（像不像博主）

- [ ] 缩略图小到指甲盖，还能看清 **3～8 个字**吗？
- [ ] 像 **小红书笔记** 还是像 **公司公众号**？
- [ ] 有没有 **大蓝渐变 / 西装 / 抽象科技球**？有就重做
- [ ] 和文章主题 **一眼相关** 吗？（AI/周报/工具）
- [ ] 是否留了叠字区或字已清晰嵌入？
