# 封面知识库 · 小红书万赞封面画风库（附生图提示词）

> 来源：万赞图文笔记常见版式归纳 + 信息图爆款结构。给 IMAGE Agent 直接检索套用。

## 一、2025～2026 小红书高频封面版式

| 版式            | 特征                      | 适合内容           |
| --------------- | ------------------------- | ------------------ |
| **纯色大字报**  | 1 个背景色 + 超大标题     | 干货、避坑、清单   |
| **备忘录体**    | 仿 Notes 界面、便签纸     | 步骤教程、AI 设置  |
| **阅读卡片**    | 白卡片 + 黑字 + 少量 icon | 知识科普           |
| **多巴胺色块**  | 高饱和撞色、几何色块      | 年轻向、工具推荐   |
| **奶油/莫兰迪** | 低饱和温柔色              | 生活方式、效率美学 |
| **对比拼图**    | 2～4 格 before/after      | 改造、效率对比     |
| **真人+大标题** | 自拍/半身 + 侧边大字      | 口播向、经验分享   |

**共同规则**：3:4 竖版；主标题 **3～10 字**；缩略图可读；**不像企业海报**。

## 二、AI 效率赛道 · 6 套可直接用的画风

### 画风 1：番茄红大字报

- 背景：米白 `#FFF8F0`
- 主字：黑色超粗「别再\_\_\_」
- 强调：番茄红底白字标签「AI 实测」

```
小红书大字报封面，3比4竖版，米白色背景，超大黑色粗体字「别教AI写周报」，
下方番茄红色块白字小标签「8分钟实测」，右下角可爱emoji电脑贴纸，
整体接地气博主风格，禁止商务蓝和企业风。
```

### 画风 2：柠檬黄高亮备忘录

- 仿浅黄便签纸，两行标题，荧光笔划线

```
小红书备忘录封面，3比4，浅黄色便签纸纹理，手写感黑色标题
「AI学会我操作了」，第二行较小字「下周一键跑」，黄色荧光笔划线强调，
左上角红色图钉贴纸，清新手账风，禁止PPT商务风。
```

### 画风 3：薄荷绿阅读卡片

- 中央白色圆角卡片，四周薄荷绿留白

```
小红书阅读卡片封面，3比4，薄荷绿柔和背景，中央白色圆角卡片，
卡片内黑色粗体标题「打工人AI效率」，下方3个emoji图标一排：💻☕️⚡️，
简约可爱信息图风格，多留白，禁止科技蓝渐变。
```

### 画风 4：左右对比（效率类）

```
小红书对比封面，3比4，左右分屏，左半边灰调乱桌面文字「40min」，
右半边暖光清爽桌面文字「8min」，中间黄色闪电符号，卡通手绘风，
生活场景非商务办公室。
```

### 画风 5：俯拍书桌（生活美学）

```
小红书生活风封面，3比4，俯拍木质书桌，笔记本电脑和手机，
一杯拿铁，自然窗光，温暖治愈，上方留白区域用于叠加标题，
整体像博主实拍美学，不要广告棚拍感。
```

### 画风 6：多巴胺撞色（工具种草）

```
小红书多巴胺封面，3比4，粉橙撞色几何色块背景，白色超大字「OpenClaw实测」，
黑色描边增强可读，一个可爱龙虾简笔画贴纸，年轻活力，非企业宣传。
```

## 三、标题在图中的排版

- 主标题：**60pt 级别视觉重量**（占画面 1/3 高）
- 副标题：主标题 50% 大小
- 中文标题用 **「」** 包裹便于模型渲染
- 全图汉字 **不超过 15 个**（字多就后期叠，别塞满）

---

## 四、必禁清单（导致「很难受的商务感」）

1. 深蓝 / 蓝紫渐变 + 白色细字
2. 抽象地球、网络节点、区块链线条
3. 穿西装的人微笑点头
4. 「数字化转型」「赋能」「解决方案」视觉
5. 玻璃办公楼、会议室、演讲台
6. 像 Canva 默认 **Business** 模板的一切
