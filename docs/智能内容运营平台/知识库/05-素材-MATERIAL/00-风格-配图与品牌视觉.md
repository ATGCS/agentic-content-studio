# 素材知识库 · 正文配图风格（生活感信息图，不要商务插图）

## 一、正文配图 ≠ 封面，但同一套审美

正文 `[[IMAGE:fig-1]]` 配图常比封面更「商务」——因为模型爱画：**等距办公插画、蓝色图标、抽象数据图**。

**目标**：配图像小红书 **笔记内页图**——手绘信息图、步骤卡片、对比小图、屏幕截图风。

---

## 二、账号视觉规范（AI 效率赛道）

| 维度 | 采用                             | 严禁                       |
| ---- | -------------------------------- | -------------------------- |
| 画风 | 手绘卡通 / 扁平插画的 **生活版** | 企业 Isometric、商务 stock |
| 背景 | 莫兰迪、奶油、浅粉、薄荷绿       | 深蓝、冷灰 corporate       |
| 线条 | 圆润、略涂鸦、便签感             | 锐利科技线条、3D 玻璃      |
| 人物 | Q 版打工人、手部特写、背影       | 西装演讲、握手、团队合影   |
| 图标 | emoji、简笔贴纸、马克笔圈重点    | Font Awesome 商务图标墙    |
| 文字 | 关键词短句、数字大、分点清晰     | 密密麻麻小字说明书         |

**固定配色（全号统一）**：

- 背景：奶油米白 `#FFF8F0` 或浅杏 `#FFFAF5`
- 强调：番茄红 `#E85D4C` 或柠檬黄 `#FFD93D`（二选一）
- 正文色：深灰 `#333333`，不用纯黑

## 三、正文配图 5 种类型（按内容选）

### 1. 步骤卡片（教程类）

- 竖版小卡片，「步骤 1/2/3」+ 简笔画
- 每步 **一个动作** + 一句口语说明
- 例：「打开 Excel → 让 AI 录操作 → 下次一键跑」

### 2. 对比图（Before / After）

- 左右分屏，**生活场景**对比（乱桌面 vs 清爽桌面）
- 禁：商务增长曲线配西装人

### 3. 清单/checklist 图

- 便签纸或备忘录 UI，❌✅ 列表
- 适合「3 个避坑」「5 个设置」

### 4. 屏幕氛围图（工具类）

- **俯拍角度** 笔记本 + 手机，屏幕内容 **模糊可读**
- 自然光、木质桌面、咖啡杯—— **像博主实拍**
- 禁：悬浮全息 UI、科幻 HUD

### 5. 金句/单点图（中段防划走）

- 一句钩子 + 荧光笔划线 + 大量留白
- 用于正文「微转折」配图

## 四、与正文段落的对应关系

REWRITE 写的 `imageSlots.prompt` 必须 **具体可画**：

| ❌ 模糊 prompt   | ✅ 可画 prompt                                                                                                        |
| ---------------- | --------------------------------------------------------------------------------------------------------------------- |
| 一张 AI 科技插图 | 俯拍木桌笔记本电脑，屏幕显示 Excel 表格，旁边黄色便签写着「一键周报」，手绘箭头指向保存按钮，奶油色背景，小红书手绘风 |
| 效率提升概念图   | 左右对比：左边乱糟糟堆满文件的桌面，右边只有一台笔记本和咖啡杯，卡通风格，莫兰迪配色                                  |
| 商务办公场景     | Q 版打工人趴在桌上对着电脑叹气，头顶对话框「又要写周报」，可爱手绘，浅粉背景                                          |

**每个 slot 只表达 1 个信息点**，不要一张图塞 5 个概念。

## 五、Seedream 正文配图提示词模板

### 步骤图

```text
小红书风格手绘信息图，竖版3比4，奶油米白背景，可爱卡通简笔画。
画面展示3个步骤卡片从上到下排列：步骤1「让AI看你的操作」配笔记本简笔画，
步骤2「保存为技能」配便签贴纸，步骤3「下次一键执行」配闪电emoji。
手绘马克笔风格标题「周报自动化」，圆润线条，莫兰迪浅粉浅绿点缀。
禁止：商务isometric、企业蓝、西装、会议室、科技球体、复杂数据大屏。
```

### 对比图

```text
手绘卡通分屏对比图，竖版，左边标题「以前40分钟」画面是乱桌+叹气小人，
右边标题「现在8分钟」画面是清爽桌面+咖啡+完成勾选，中间黄色闪电分隔。
配色奶油底+番茄红强调，像博主笔记插图，非广告海报。
禁止：商务风格、写实摄影、企业宣传构图。
```

### 氛围/工具图

```text
俯拍视角生活场景插画，木质书桌，一台笔记本电脑屏幕微亮，
旁边手机显示聊天界面，一杯拿铁，窗外柔和日光。
整体温暖治愈，小红书生活美学，轻微手绘质感而非照片级写实。
禁止：stock photo感、商务办公室、冷色调科技风。
```


## 七、和封面的关系

- **封面**：抓点击，字大、信息少
- **配图**：帮理解，可稍密但仍是 **卡片化、分点化**
- 同一篇：封面用大字报，配图用同色系手绘， **不要封面生活、配图商务**
