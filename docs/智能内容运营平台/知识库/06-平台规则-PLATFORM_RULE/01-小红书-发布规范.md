# 平台规则 · 小红书发布规范

## 一、内容规格

| 项   | 规范                             |
| ---- | -------------------------------- |
| 标题 | 建议 ≤ 20 字，可含 1～2 个 emoji |
| 正文 | 600～1200 字为佳                 |
| 标签 | 5～10 个，混合大中小流量词       |
| 图片 | 首图决定点击；比例 3:4 或 1:1    |

## 二、发布前检查清单

- [ ] 无导流外链、微信号硬广
- [ ] 无绝对化用语（「最好」「第一」需有依据）
- [ ] 图片清晰，无大面积水印
- [ ] 标签与内容主题相关，不蹭无关热点

## 三、排版建议

- 段落短、空行多
- 重点用 emoji 或小标题，不用纯大段
- 首图与标题信息一致

## 四、手动发布流程（无 OAuth 时）

1. ACS 审核通过 → 发布管理导出发布包
2. 复制标题、正文、标签
3. 上传封面与配图
4. 发布后回 ACS 记录链接（如有）

## 五、常见违规

- 医疗功效宣称、金融收益承诺
- 未授权搬运、洗稿
- 诱导私信、站外交易
