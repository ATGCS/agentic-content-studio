# 平台规则 · 抖音发布规范

## 一、内容规格

| 项        | 规范                                |
| --------- | ----------------------------------- |
| 视频时长  | 干货类 60～180 秒；口播可 30～60 秒 |
| 标题/描述 | 描述区可放补充信息 + 话题           |
| 话题      | 2～5 个，1 个主话题 + 若干长尾      |
| 封面      | 竖版 9:16，人脸/产品清晰            |

## 二、发布前检查清单

- [ ] 前 3 秒有钩子
- [ ] 字幕准确（口播建议开字幕）
- [ ] 无未授权 BGM、影视片段
- [ ] 描述区无违规导流

## 三、描述区模板

```text
【一句话钩子】

【3 个要点关键词】

#主话题 #长尾1 #长尾2
```

## 四、手动发布流程（无 OAuth 时）

1. ACS 导出抖音发布包（文案 + 封面）
2. 按文案录制或剪辑视频
3. 上传并粘贴描述与话题
4. 定时发布选流量高峰（早 7-9 / 午 12-14 / 晚 19-22）

## 五、常见违规

- 虚假夸张、标题党与内容不符
- 未标注 AI 生成内容（平台政策变化时需关注官方公告）
- 站内互导其他平台
