# 标签知识库 · 标签策略

## 一、标签的作用

- 小红书：**搜索 + 推荐** 双重流量
- 抖音：**话题聚合** 与冷启动

REWRITE Agent 会在平台改写时生成 tags；本文供规则参考。

## 二、组合原则

| 类型   | 数量 | 说明                           |
| ------ | ---- | ------------------------------ |
| 核心词 | 1～2 | 与内容主题强相关               |
| 场景词 | 2～3 | 受众场景（职场、宝妈、AI工具） |
| 长尾词 | 2～4 | 竞争较小、精准搜索             |
| 热点词 | 0～1 | 仅当内容与热点真正相关         |

**总数**：小红书 5～10 个；抖音 2～5 个话题。

## 三、示例（AI 效率主题）

**小红书**

`AI工具` `职场效率` `打工人` `周报模板` `自动化办公` `OpenClaw` `Hermes`

**抖音**

`#AI工具` `#效率提升` `#职场干货`

## 四、避坑

- 不要堆砌无关大流量词（限流风险）
- 同一账号标签体系保持相对稳定
- 品牌词、系列词可固定出现在每篇

## 五、维护

每月从数据复盘更新「高表现标签池」追加到本文末尾。
