# 账号风格 · 人设填写指南

## 一、说明

系统内 **账号管理 → profile** 是 AI 生成时的主要风格输入。  
本文档用于团队对齐人设，填写后可同步到 profile 字段。

## 二、人设卡片（复制填写）

```text
【账号名称】
【一句话定位】例：帮打工人用 AI 省时间的效率博主
【目标受众】年龄/职业/痛点
【口吻】例：亲切、略幽默、不贩卖焦虑
【常用句式】例：「你是不是也…」「我实测了一下…」
【禁用】例：不说「躺赚」、不夸大 AI 替代人类
【代表栏目】AI工具 / 职场 / 生活
【参考账号】（可选，仅内部参考）
```

## 三、口吻示例

**推荐**

> 我上周试了一下，说实话有点超预期…

**避免**

> 本产品行业领先，全面赋能用户…

## 四、多账号差异

若运营多个账号，每人设单独一篇文档，文件名如：

`01-账号-效率工具号.md`

## 五、与知识库的关系

- 口吻、禁忌 → 优先写进 **账号 profile**
- 方法论、案例 → 写进 BODY / TOPIC 库
- 本文档变更后，记得更新系统内账号配置
