# 开发排期 V1

周期：**8 周 / 4 Sprint**，每 Sprint 2 周。人力按 **1 全栈 + 1 前端** 估算，可按实际调整。

## Sprint 1（第 1–2 周）：基础与内容资产

| 任务 | 产出 | 依赖文档 |
| ---- | ---- | -------- |
| Monorepo 脚手架 | apps/web、apps/api、packages/db | 开发规范 V1 |
| Prisma migrate + seed | 可运行 SQLite | 数据库设计 V1 |
| Auth + RBAC 中间件 | 登录、/auth/me | 权限模型 V1 |
| 选题 CRUD API + 页面 | Topics | API / 页面原型 |
| 内容 CRUD API + 页面 | Content Center | 同上 |
| 平台账号同步（Adapter 骨架） | POST /accounts/sync | TurboPush 二开 V1 |
| 账号画像 API + 页面 | Account Profile | 同上 |

**Sprint 1 完成标准**：可登录；可建选题与内容；可编辑账号画像；账号列表可从 Adapter mock 拉取。

## Sprint 2（第 3–4 周）：Agent Runtime + IMA

| 任务 | 产出 | 依赖 |
| ---- | ---- | ---- |
| ModelGateway（DeepSeek） | 可 chat | Agent Runtime V1 |
| PromptEngine + 种子 Prompt | 标题/正文模板 | 数据库 prompts |
| ToolEngine + ima-provider | IMA 搜索落库 | |
| TaskEngine + agent_runs | 单 Agent 执行 | |
| TITLE / BODY / REWRITE Agent | API 可调 | API 文档 |
| AI Generate 页面 | 运行 Agent、看记录 | 页面原型 |
| Prompt Center（ADMIN） | 列表与编辑 | |

**Sprint 2 完成标准**：对一条 content 完成 IMA 搜索 + 标题/正文生成 + 至少一个平台版本改写。

---

## Sprint 3（第 5–6 周）：审核 + 发布

| 任务 | 产出 | 依赖 |
| ---- | ---- | ---- |
| 内容/版本状态机 | 与 ContentStatus 一致 | 具体设计 状态机 |
| Review API + 页面 | 提交/通过/驳回 | 权限模型 |
| REVIEW Agent（辅助） | 风险提示 | Agent Runtime |

| Publishing Center 页面 | 创建任务、发布 | |

**Sprint 3 完成标准**：审核通过后可创建发布任务并拿到发布结果（mock 亦可）。
