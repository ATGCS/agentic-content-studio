# 页面原型文档 V1（低保真）

说明：按 **页面 → 区域 → 功能 → 按钮/操作** 描述，供前端 Mock 与联调。高保真视觉见《详细设计文档 V1》第九章。

---

## 1. Dashboard（工作台）

| 层级 | 内容 |
| ---- | ---- |
| **页面** | 工作台首页 |
| **区域** | 顶部统计卡片 |
| **功能** | 展示待生成、待审核、待发布、本周已发布数量 |
| **按钮** | 无 |
| **区域** | 快捷入口 |
| **功能** | 跳转选题、内容、审核、发布 |
| **按钮** | 「新建选题」「新建内容」「审核队列」「发布队列」 |
| **区域** | 最近内容列表 |
| **功能** | 最近 10 条内容，显示标题、状态、更新时间 |
| **按钮** | 行内「查看」 |

## 2. Topics（选题管理）

| 层级 | 内容 |
| ---- | ---- |
| **页面** | 选题列表 |
| **区域** | 筛选栏 |
| **功能** | 按状态、平台、创建人筛选 |
| **按钮** | 「查询」「重置」 |
| **区域** | 列表区 |
| **功能** | 表格：标题、目标平台、状态、创建人、时间 |
| **按钮** | 「新建选题」；行内「编辑」「删除」「创建内容」 |
| **页面** | 选题详情/编辑（抽屉或子页） |
| **区域** | 表单 |
| **功能** | 标题、说明、目标平台多选、来源 |
| **按钮** | 「保存」「取消」 |

## 3. Content Center（内容资产）

| 层级 | 内容 |
| ---- | ---- |

| **区域** | 右侧信息栏 |

## 4. AI Generate（AI 生成）

| 层级 | 内容 |
| ---- | ---- |
| **页面** | AI 生成工作台（可从内容详情进入） |
| **区域** | 上下文区 |
| **功能** | 展示当前 content、账号画像摘要、IMA 搜索结果 |
| **按钮** | 「重新搜索 IMA」 |
| **区域** | Agent 操作区 |
| **功能** | 选择 Agent：标题 / 正文 / 标签 / 改写 / 审核辅助 |
| **按钮** | 「运行 Agent」「查看执行记录」 |
| **区域** | 输出预览区 |
| **功能** | 展示本次 Agent 输出，支持采纳到主内容或版本 |
| **按钮** | 「采纳标题」「采纳正文」「生成平台版本」 |

## 5. Review Center（审核中心）

| 层级 | 内容 |
| ---- | ---- |
| **页面** | 审核队列 |
| **区域** | Tab |
| **功能** | 待审核 / 已通过 / 已驳回 |
| **按钮** | 无 |

| **区域** | 驳回弹窗 |
| **功能** | 填写审核意见 |
| **按钮** | 「确认驳回」「取消」 |

## 12. 页面与 API 映射

完整映射（含 sidebar 页、详情页、后端/前端联调状态、优先级）见：

**[前端页面-API映射表 V1](./前端页面-API映射表%20V1.md)**

简表：

| 页面 | 主要 API | 联调状态 |
| ---- | -------- | -------- |
| Dashboard | `GET /dashboard/stats`, `GET /contents` | 部分 wired |
| 内容项目 | `/contents`, `/versions`, `/agents/*/run` | 部分 wired |
| Agent 任务 | `GET /agent-runs` | 后端有，前端 mock |
| 审核中心 | `/reviews` | 部分 wired |
| 发布管理 | `/publishing/tasks` | 后端有，前端 mock |
| 素材库 | `/materials/library/*` | 文档 P1，均未实现 |
| 数据复盘 | `/analytics/*` | 后端有，前端 mock |
| 知识库 | `/ima/*` | wired |
| 平台账号 | `/accounts`, `/account-profiles` | 部分 wired |
| 系统设置 | `/integration/*`（P1）, `/settings`, `/users`（P2） | 均未实现 |
| Topics / Prompts | `/topics`, `/prompts` | 部分 wired |
